# FORMULARIO CON GOOGLE SHEETS

## Instrucciones

1. Abre el archivo `formulario.html` con doble clic en tu navegador.
2. Llena el formulario y presiona "Enviar".
3. Los datos se guardarán automáticamente en tu hoja de Google Sheets vinculada.

## Cómo compartir con otros:

- Puedes enviar este archivo por correo, Drive, WhatsApp o subirlo a:

  - https://netlify.com (hosting gratuito)
  - https://pages.github.com (GitHub Pages)

## ¿Qué necesitas?

- Tu Google Apps Script debe estar publicado como aplicación web:
  - Ejecutar como: Tú mismo
  - Acceso: Cualquiera, incluso anónimo
